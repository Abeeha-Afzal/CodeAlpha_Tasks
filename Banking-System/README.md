# Banking System (C++)

This program implements a **Banking Management System** in C++ using Object-Oriented Programming (OOP) concepts.  
It simulates basic banking operations such as account creation, transactions, and account management.

## **Task Description**

The program performs the following operations:

- Creates a new customer account  
- Stores account number and balance  
- Allows **deposit and withdrawal** operations  
- Maintains transaction history  
- Searches customers using account number  
- Displays complete account details  
- Provides a menu-driven interface for user interaction  

## **Program Structure**

The program is organized using multiple classes:

### **Transaction**
Stores transaction type and amount.

### **Account**
Handles account details including:
- Account number  
- Balance  
- Transaction history  
- Deposit and withdrawal operations  

### **Customer**
Stores customer information and links it with account details.

### **Main Function**
Controls program flow and provides menu-based interaction.

## **OOP Concepts Used**

- **Encapsulation** – Data and functions are organized inside classes  
- **Abstraction** – Banking operations are handled through class methods  
- **Composition** – Account contains transaction history  
- **Vectors** – Used to manage multiple customers  

## **Features**

- Create and manage multiple accounts  
- Deposit and withdraw money  
- Maintain transaction history  
- Search accounts efficiently  
- Menu-driven user interface  

## **Language Used**

- **C++**
