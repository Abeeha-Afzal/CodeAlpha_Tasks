# Login System (C++)

This program implements a **simple login and registration system** using file handling.  
It demonstrates **Object-Oriented Programming (OOP)** concepts in C++ along with basic authentication logic.

## **Task Description**

The program performs the following operations:

- Takes **username and password** input from user  
- Registers new users and stores credentials in a file  
- Allows existing users to **log in** using saved credentials  
- Reads and verifies user data from file  
- Displays **success or failure messages** based on login validation  

## **Program Structure**

The program is organized using functions:

### **Register_User**
Accepts username and password and stores them in a file.

### **login_User**
Reads stored data and verifies user credentials.

### **Main Function**
Controls program flow and calls registration and login functions.

## **Concepts Used**

- **File Handling** – Uses `ifstream` and `ofstream`  
- **Functions** – Separate logic for registration and login  
- **Strings** – Used for handling usernames and passwords  
- **Conditional Logic** – Used for authentication validation  

## **Features**

- **User Registration system**  
- **User Login verification**  
- **File-based data storage**  
- **Simple and interactive console interface**  

## **Language Used**

- **C++**
